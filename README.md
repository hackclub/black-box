**Black Box** was a [You Ship, We Ship](https://ysws.hackclub.com) event about *constraints*.\
Write **an interactive program** for an **8x8 LED matrix**, get *a portable device to run it on.*

This event ran from **February 22** at 10AM EST to **March 29** at 11:59PM EST.
